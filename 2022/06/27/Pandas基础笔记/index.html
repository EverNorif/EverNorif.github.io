小入门

  • 数据结构：series，DataFrame

  • .info()：信息查看

  • .shape：行列数目

  • .dtype：查看数据类型

  • .index：显示索引名（行）

  • .columns：显示列名

  • .describe()：查看统计摘要

  • 选择

    • 普通选择：
      • df['列名']：取列
      • df[xx:xx]：取行
    • .loc[行信息,[列信息]]
    • .iloc[]：只接受整数
    • 布尔索引：
      • df[判断条件]：返回DataFrame
      • 多条件筛选：用(条件1)&(条件2)
      • isin()：传一个列表

  • astype()：强转类型

  • .sum()：求和。指定列应该在调用.sum()之前

  • 判断空值

    • 1)判断数值是否为空用 pd.isna,pd.isnull,np.isnan
      2)判断字符串是否为空用 pd.isna,pd.isnull；
      3)判断时间是否为空用 pd.isna,pd.isnull,np.isnat

  • 去重：

    • .drop_duplicates(['根据某一列'])

读取数据

数据类型 说明 读取方法

excel xis或者xlsx文件 pd.read_excel

    • engine参数

      df = pd.read_csv(

一些常见操作：

  • .head():查看前几行数据
  • .shape：返回数据的形状（行，列）
  • columns：返回列名列表
  • index：返回索引列（每一行的索引）
  • dtypes：查看每一列的数据类型

数据结构 DataFrame和 Series

  • DataFrame：二维表格，有行有列

    • 列：columns
    • 行：index

  • Series：一维数据，代表一行或者一列

    • 如果查询得到了一维，会进行自动降维

  • .Series()：产生Series对象

    • 可以传入列表
    • 可以传入字典（字典的key变成index）
    • 默认使用0开始的index索引
      • 可以使用index=[...]参数来指定
    • 根据标签查询Series（类似python中的字典）
      • s['index']：返回原生数据类型
      • s['index1','index2',...]：返回Series

  • .DataFrame()：产生DataFrame对象

    • 可以传入字典，字典的每一个值是一个列表。

    • ```python
      df =pd.DataFrame(data=data,index=[…])

  • df.apply：沿着DataFrame的某一个轴执行函数，相当于每次传入了一个Series

    • axis=0：沿着index，Series的index是DataFrame的index
    • axis=1：沿着columns，Series的index是DataFrame的columns

  • df.assign

    • assign可以一次新增多列，apply一次新增一列
    • 只能添加列，即（axis=1），不能沿index操作
    • 返回一个新对象，包含新增列和原来的列

  • 按照条件选择分组分别赋值

    • 先创建空列，然后再用条件表达式分别选中不同的行，最后分别赋值

    • ```python
      df[‘aa’] = ‘’ # 不创建空列也可以
      df.loc[df[‘vv’]>0,’新增’] = xxx
      df.loc[df[‘vv’]<=0,’新增’] = yyy

  • left,right:要merge的dataframe或者有name的Series

  • how:join类型，有‘left’，‘right’，‘outer’，‘inner’

    • inner：保留重合数据
    • outer：数据全部保留
    • left：完全保留左边数据
    • right：完全保留右边数据

  • on:join的key，left和right都需要有这个key

  • left_on:left的df或series的key

  • right_on:right的df或series的key

  • left_index,right_index:使用index而不是普通的column做join

  • suffixes:两个元素的后缀，如果列有重名，自动添加后缀

对齐关系：

  • 一对一：关联的key是唯一的
  • 一对多：左边key唯一，右边key不唯一
  • 多对多：左右两边都不唯一
    • 后面两种类型数据会被复制
    • 进行笛卡尔积

数据Concat合并

  • 使用场景：

    • 批量合并相同格式的Excel，给DataFrame添加行、添加列
    • 上下拼接或者左右拼接

  • 简单说明concat语法

    • 使用某种合并方式（inner/outer）
    • 沿着某个轴向（axis=0/1）
    • 把多个pandas对象（DataFrame/Series）合并成一个

  • concat语法

    • ```python
      pd.concat(objs,axis=0,join=’outer’,ignore_index=False)
      • other：单个DataFrame，Series，dict或者list

实现分组数据统计

groupby：先对数据进行分组，然后在每个分组上应用聚合函数、转换函数

  • groupby('columns')：原来的columns变成索引，以此看其他列

  • groupby(['columns1','colunms2',...])：形成多级索引（分层索引）

    • as_index=False：指定不要形成多级索引

  • 同时查看多种数据统计：

  • df.groupby('A').agg([np.sum, np.mean, np.std])

  • groupby得到数据结构可以迭代

  • 按照name来区别不同的group

分层索引MultiIndex

MultiIndex：

  • 分层索引：在一个轴向上拥有多个索引层级，可以表达更高维度数据的形式

  • 可以更方便的进行数据筛选，如果有序则性能更好

  • groupby等操作的结果，如果是多key，结果是分层索引

  • 一般不需要自己创建分层索引（MultiIndex有构造函数但是一般不用）

  • 分层索引的筛选利用元组的形式。

    • 也可以只用一个，但是有索引的先后顺序

  • unstack()：将多层索引变成列

  • 在选择数据的时候：

    • 元组(key1,key2)代表筛选多层索引，key1是一级索引，key2是二级索引
    • 列表[key1,key2]代表同一层的多个key，key1和key2是并列的同级索引

  • slice(None)：代表筛选这一索引的所有内容

  • reset_index()：将多级索引变成普通的列

map，apply，applymap数据转换函数

大致讲解：

  • map：只用于Series，实现每个值到值的映射

  • apply：用于Series实现每个值的处理，用于DataFrame实现某个轴的Series的处理

  • applymap：只能用于DataFrame，用于处理DataFrame的每个元素

  • Series.map()

    • 可以传递字典或者函数
    • 字典中即映射关系
    • 函数：将值传入作为参数，转化成返回值

  • apply：前面说过，对每个元素（元素或者Series）进行函数操作

  • applymap：传入函数，对于函数，函数的参数是DataFrame每一个元素

实现groupby每个分组的apply

知识：pandas的groupby遵从split，apply，combine模式

  • 首先根据groupby的条件将大表切分成多个小表
  • 对每一个小表进行函数的应用得到新小表
  • 然后将多个小表拼接

将列式数据变成二维交叉形式，便于分析，叫做重塑或透视。

unstack()和stack()互逆

  • unstack：把index变成columns

pivot()：相当于对df使用set_index创建分层索引，然后调用unstack

  • 简化的方法实现了unstack

  • stack:

    • ```python
      df.stack(level=-1, dropna=True)
      • 与stack互逆，将index变成column

  • pivot：

    • ```python
      df.pivot(index=None, columns=None, values=None)
